1. 文档概述

1.1 产品简介

星际娱乐是一款创新性的"娱乐+社交+电商"平台，核心通过付费会员体系与多级分佣机制，连接娱乐产业上下游资源（艺人、企业、粉丝、商家），打造一个集人才孵化、内容展示、资源对接、票务服务、消费变现于一体的闭环生态。

1.2 产品目标

  • 短期: 快速获客，通过分享经济机制驱动用户增长，完成核心功能建设，重点验证票务与会员分佣模式。
  • 中期: 提升平台活跃度，沉淀优质内容与专业用户（商家、导演、艺人），拓展B端合作伙伴，验证各VIP服务的商业化价值。
  • 长期: 成为娱乐产业的基础设施与流量入口，重塑行业资源分配方式。

3.1.2 演唱会模块

子功能模块 优先级 详细需求说明
列表页 P0 1. 展示: 以城市（如深圳站、南宁站）为维度展示演唱会列表，包含海报、名称、时间、地点。

3.1.3 好声音赛事模块

子功能模块 优先级 详细需求说明

2. 排序: 按票数或热度排序。
3. 搜索: 可按编号、姓名搜索选手。
投票机制 P0 1. 免费投票: 每个用户每天可免费投1票（需验证账号）。
2. 付费投票: 用户可充值购买票数（如100元购买1000票），为选手加油。后台可配置充值档位与票数换算比例。

2. 支付后填表: 用户支付成功后，才跳转至详细的报名信息表单页面进行填写。
3. 信息审核: 提交后进入后台审核流程。
地区分赛区 P1 1. 展示: 首页应展示全国各地分赛区的活动视频剪辑或宣传图。
2. 发布: 分赛区内容由战略合作伙伴（地方娱乐公司）在后台上传发布。

4. 非功能性需求

  • 性能: 首页及核心页面加载时间低于2秒，票务抢购场景下系统稳定。
  • 安全: 用户数据加密，接口防刷（投票、抢票、注册、分佣），支付密码校验，后台操作日志完备。加强对实名认证信息的管理。
  • 兼容性: 支持主流iOS和Android机型上的微信小程序平台。
  • 可扩展性: 系统设计需考虑未来接入更多第三方票务平台、支付渠道的可能性。

5. 项目规划与里程碑

5.1 版本发布计划 (修订)

  • V1.0 (MVP版本):
    • 核心功能: 会员体系（支付+分佣）、演唱会票务（创建、购票、实名、分佣）、好声音赛事（付费报名、投票）、基础后台（审核、配置、分佣管理）。
    • 目标: 跑通"票务-会员-分佣"这个最核心的商业闭环。
  • V1.1 (B端拓展版):
    • 新增功能: 战略合作伙伴独立后台、商家会员入驻与推广系统、导演编剧VIP及其发布功能。
    • 优化: 增强后台管理能力，支持多地活动运营。
  • V1.2 (内容生态版):
    • 新增功能: 直播功能、社区电商、更丰富的UGC内容互动。
    • 优化: 性能与体验优化，数据统计分析工具。

5.2 优先级定义

  • P0: 必须上线，否则MVP无法运行（如票务、会员、分佣、支付）。
  • P1: 重要功能，对核心业务有重大补充或能快速验证B端需求（如合作伙伴后台、商家推广、付费投票）。
  • P2: 锦上添花的功能，或需要验证需求的功能（如直播、复杂电商），可后续迭代。

6.2 关键业务规则

  1. 佣金生效原则：佣金只在推荐人本人是会员期间，其下级发生消费行为时产生。
  2. 关系链锁定：用户通过某分享链接注册后，其关系即永久锁定在该分享者名下，不受后续双方会员状态变化影响。
  3. 原创版权免责：平台提供原创承诺协议流程，明确侵权责任由发布者自负，平台不承担初步审核责任。
  4. 会员到期处理：到期后所有与收益相关的权限立即终止，数据看板功能关闭，直至续费。
  5. 特殊退款规则：如"歌手/演员"等特殊会员因平台未在约定时间（如3个月）内安排演出而申请退款，退款成功后，其会员资格及整个下级关系网络将被永久清空。

7. 【核心】原创音乐人中心

7.1 作品发布与管理 (FR3.1)

  • 词曲创作人可发布作品，需填写：歌曲名称、词曲作者、简介、封面图、音频文件、出售价格。
  • 强制协议：发布前必须勾选《原创版权承诺协议》，承诺作品原创，侵权责任自负。
  • 支持设置试听片段（默认15-30秒）。
  • 支持"公开/隐藏"状态切换。作品售出后，卖家可设置为"隐藏"。
  • 支持创作者上传身份证进行实名认证。

7.2 作品展示与交易 (FR3.2)

  • 前端仿照QQ音乐风格打造精美展示页面，包含"新曲推荐"、"排行榜"等板块。
  • 集成电子签约功能，购买流程化。平台收取交易额的10%作为佣金。
  • 防私单机制：限制非好友私信功能。在显著位置提示"平台外交易不受保护，且一经发现将封禁账号"。

7.3 个人主页 (FR3.3)

  • 为每位词曲创作人生成独立的个人主页，展示个人简介、代表作、全部作品列表。

8. 【核心】原创音乐大舞台

8.1 活动管理 (FR4.1)

  • 平台运营端可发布每周六的"原创音乐大舞台"活动，并设置每周不同的主题（如：古风周、情歌周）。

8.2 歌手与作品对接 (FR4.2)

  • 系统可将海选歌手的资料与词曲创作人上传的作品进行匹配推荐。
  • 歌手可报名参加活动，演唱指定的原创歌曲。

8.3 品牌宣传 (FR4.3)

  • 此板块是平台核心亮点，宣传语突出："在这里听到全网找不到的原创好声音"。

9. 投票系统

9.1 投票规则 (FR5.1)

  • 为"寻找好声音"等赛事活动提供投票功能。
  • 初始虚拟票数：为防止冷启动，系统为每位选手随机生成50-200的初始虚拟票数。真实票数超过后，显示真实票数。
  • 普通用户：每日拥有固定免费投票权（如10票）。
  • 会员激励：平台会员每日可获得更多投票权（如50票）。
  • 裂变激励：用户通过分享链接邀请新用户注册投票，每成功邀请1人可额外获得投票权（如10票）。

9.2 显示与公平 (FR5.2)

  • 前端仅显示总票数，不显示投票人列表。

10. 推广员与商家管理系统

10.1 推广员管理 (FR6.1)

  • 后台可创建和管理"推广员"角色。
  • 推广员任务与收益：设定月度推广商家目标（如30家）。完成可获得高额提成（如20%）及固定底薪奖励（如3000元）。

10.2 商家入驻 (FR6.2)

  • 商家需填写完整信息（营业执照、地址等）并付费入驻。
  • 入驻后自动生成专属推广二维码和可自定义的推广海报。
  • 商家收益：顾客通过扫描商家二维码注册、消费，商家可获得佣金。

10.3 智能海报生成 (FR6.3)

  • 提供在线工具，商家可选择模板，在线编辑门店信息并嵌入二维码，生成个性化宣传海报。

11. 电子签约系统

11.1 功能说明 (FR9.3)

  • 集成电子签约服务。流程为：用户付款 -> 自动生成标准合同 -> 用户填写信息 -> 在线签署 -> 合同归档。

12. 后续规划（V3.0）

  1. 虚拟商品/电商商城：上线平台周边、数字专辑等虚拟商品，扩展盈利模式。
  2. 直播功能：集成直播，用于线上海选、演出直播、粉丝互动。
  3. IP孵化与版权管理：深度介入优质IP的孵化、版权管理和分销。
  4. 金融服务深化：与合规金融机构合作，将"企业援助"功能产品化、规范化。
  5. 独立APP开发：脱离微信生态，开发独立APP，实现更复杂的功能和更好的用户体验。
